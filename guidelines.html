Submission Guidelines

Thank you for your interest in submitting your CSS art to our gallery! Please read the following guidelines carefully before submitting.

General Requirements

  • All artwork must be created entirely using CSS (HTML is allowed for structure).
  • Submissions must be original and not infringe on any copyrights.
  • Artwork should be visually appealing and demonstrate creative use of CSS.
  • Keep your CSS code clean, well-organized, and commented.

Technical Guidelines

  • Provide the complete CSS code for your artwork.
  • Specify any HTML structure required for the CSS to render correctly.
  • Ensure your code is compatible with modern web browsers (Chrome, Firefox, Safari).
  • Optimize your CSS for performance (avoid excessive or inefficient code).

Submission Process

  1. Create an account or log in to the website.
  2. Go to the Submit Art" page.
  3. Fill out the submission form with the required information (title, CSS code, etc.).
  4. Review your submission carefully before submitting.

Content Restrictions

We reserve the right to remove from our website submissions that are:

  • Offensive, discriminatory, or harmful in any way.
  • Sexually suggestive or exploit, abuse, or endanger children.
  • Unrelated to CSS art or of poor quality.

By submitting your artwork, you agree to abide by these guidelines. We look forward to seeing your creative CSS masterpieces!
